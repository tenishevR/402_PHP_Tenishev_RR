# Калькулятор — Slim Framework (ЛР3)

Веб-приложение "Калькулятор" на микрофреймворке **Slim** с **REST API** и **одностраничным интерфейсом (SPA)**.

## Запуск приложения

1. Перейдите в каталог `Task03`:
   ```bash
   cd Task03
2. Установите зависимости Composer (если ещё не установлены):
    ```bash
    composer install
3. Запустите встроенный веб-сервер PHP:
    ```bash
    php -S localhost:3000 -t public
4. Откройте в браузере:
    ```bash
    http://localhost:3000/
